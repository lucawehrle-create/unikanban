## Hinweise

- **Text final prüfen:** KI-generierter Text kann minimal abweichen — vor dem
  Posten gegenlesen. (Aktuelles Set wurde geprüft: Rechtschreibung korrekt.)
- **Konsistenz halten:** Gleiche Schrift/Farben für künftige Posts → einheitlicher
  Feed. Bei eigener Bearbeitung Canva-Vorlage mit denselben Werten anlegen.
- **Echte App-Screenshots** schlagen für Feature-Posts oft generierte Mockups —
  für Reels (Quick-Capture, Lernplan, Stundenplan) echte Screen-Recordings nutzen.
