# Profil-Setup

## Handle (Username)

Prioritäten: kurz, merkbar, „SemBan" enthalten, frei verfügbar prüfen.

1. **@semban.app** ← Empfehlung (klar als App erkennbar, sauber)
2. **@semban** (falls frei — ideal)
3. **@semban.studium**
4. **@getsemban**
5. **@semban.uni**

> Tipp: Gleichen Handle auch auf TikTok/X sichern, selbst wenn du dort (noch)
> nicht postest — Konsistenz & Schutz.

## Profilname (das fettgedruckte Feld — wird mitgesucht!)

Nicht den Handle wiederholen. Keywords reinpacken:

> **SemBan · Semesterplaner**

oder

> **SemBan — Uni-Planer & Lernplan**

## Bio (max. 150 Zeichen)

Variante A (Nutzen-fokussiert, empfohlen):
```
🎓 Der Planer, der den Uni-Rhythmus versteht
📋 Fristen · Lernplan · Stundenplan · Noten
✨ Werbefrei · offline · kostenlos
👇 Jetzt starten
```

Variante B (Community-fokussiert):
```
🎓 Wir bauen den Uni-Planer, den wir selbst wollten
📋 Alle Kurse, Fristen & Klausur-Lernpläne an einem Ort
💬 Sag uns, was dir im Semester fehlt 👇
```

## Link in Bio

- Direkt die App-URL setzen (kein Linktree nötig, solange es nur ein Ziel gibt).
- Sobald mehrere Ziele (App / Feedback / KI-Coach-Warteliste): schlankes
  Bio-Link-Tool oder eine simple Landing mit 2–3 Buttons.
- **UTM-Tracking** anhängen, um IG-Traffic zu messen, z.B.:
  `https://semban.app/?utm_source=instagram&utm_medium=bio`

## Profilbild

- App-Icon / „S"-Logo auf indigo Hintergrund. Muss als kleiner Kreis lesbar sein
  → starker Kontrast, kein Kleingedrucktes. (Nutze `visuals/profilbild-*`.)

## Story-Highlights (Struktur)

Einheitliche Cover (clean, indigo/cream, ein Icon pro Highlight):

1. **Start hier** — Was ist SemBan? (Onboarding-Stories)
2. **Features** — Lernplan, Stundenplan, Noten, Quick-Capture
3. **Tipps** — beste Study-Hack-Stories gesammelt
4. **Updates** — Build-in-Public, Roadmap, neue Features
5. **Ihr fragt** — FAQ aus DMs („Kostet das was?", „Brauche ich ein Konto?")

## Account-Typ

- Auf **Business/Creator-Konto** umstellen → Insights, beste Zeiten,
  Link-Klicks messbar. (Creator = etwas mehr Community-Tools.)

## Erste 3 Posts vor dem Bewerben (Profil darf nicht leer wirken)

1. „Was ist SemBan?" — Vorstellung (Carousel).
2. Ein starker Study-Hack (Mehrwert sofort sichtbar).
3. Ein Feature im Einsatz (z.B. Lernplan-Reel).
